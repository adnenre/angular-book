Angular is a platform and framework for building single‑page client applications using HTML and TypeScript. Its history is marked by a complete rewrite from the original AngularJS (Angular 1) to the modern Angular (2+).

- **AngularJS (1.x)** – released in 2010, introduced two‑way data binding, directives, and dependency injection. It used JavaScript and was based on the concept of scopes and controllers.
- **Angular 2+** – a complete rewrite released in 2016. It adopted TypeScript, a component‑based architecture, and a more modular design. The framework is now versioned regularly (4, 5, …, 17, 18). Major milestones include the introduction of the Angular CLI, the Ivy renderer (v9+), and standalone components (v14+).
- **Latest Versions** – As of 2026, Angular v18/19 continues to improve performance, developer experience, and aligns with modern web standards (signals, esbuild, Vite).
